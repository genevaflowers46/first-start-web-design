# First Step Web Design

A clean, modern website built to showcase affordable web design services for small businesses.

## 🌐 Live Site
https://firststepwebdesign.netlify.app/

---

## 📌 Overview

First Step Web Design is a simple, client-focused website designed to help small business owners take their first step online.

The goal of this project is to provide a clear, professional layout that shows services, pricing, and an easy way for potential clients to get in contact.

---

## 🚀 Features

- Responsive design (mobile-friendly)
- Clean, modern layout
- Service and pricing sections
- Contact form powered by Netlify Forms
- Dark mode toggle (JavaScript)
- Simple, client-focused messaging

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Netlify (deployment & forms)
- GitHub (version control)
